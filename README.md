# Cosmos — OJT Capstone Portfolio

A team portfolio site built by **COSMOS**, showcasing 6 vanilla JavaScript applications built during our OJT capstone program (Track A — Dev Agency). No frameworks, no libraries — just HTML, CSS, and JS, covering everything from async APIs to drag-and-drop boards.

🔗 **Live site:** https://adinath-21.github.io/portfolio/

## Projects

| # | Project | Highlights |
|---|---------|------------|
| 01 | Team portfolio | Data-driven cards, localStorage theme toggle, validated contact form |
| 02 | Interactive quiz | Score tracking, question state, instant feedback |
| 03 | Live news feed | NewsAPI integration, async/await, category filters, error handling |
| 04 | GitHub explorer | Multi-endpoint GitHub API, sortable repo list, language breakdown |
| 05 | Expense tracker | Full CRUD, `reduce`/`filter` totals, localStorage persistence |
| 06 | Kanban board | HTML5 Drag-and-Drop API, full board state in localStorage |

## Built with
- **HTML5** — semantic structure
- **CSS3** — Grid, Flexbox, custom properties, responsive breakpoints
- **Vanilla JavaScript** — DOM manipulation, `fetch`/async-await, Intersection Observer, localStorage, form validation

No frameworks or build tools were used — everything is plain HTML/CSS/JS.
